# 1C_Module_Price

В репозитории подготовлен единый файл модуля формы внешней обработки для 1С:

- `full_price_processing_module.bsl`

Файл уже содержит:
- заполнение товаров по отбору (СКД);
- загрузку из внешнего файла (CSV/TSV/TXT) по штрихкоду;
- заполнение старой/расчётной/новой цены;
- расчёт по формуле;
- запись документа «Установка цен номенклатуры».

Можно скопировать содержимое файла и вставить в модуль формы обработки.
