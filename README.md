# 1C_Module_Price

Репозиторий содержит описание доработки внешней обработки 1С для массового изменения цен:

- заполнение товаров по отбору и из внешнего файла/буфера;
- загрузка по штрихкоду;
- перенос старой/расчётной/новой цены в табличную часть.

См. детальный план и готовые фрагменты кода в `docs/price_processing_improvements.md`.
