# 1C_Module_Price

В репозитории подготовлен единый файл модуля формы внешней обработки для 1С:

- `full_price_processing_module.bsl`

Файл уже содержит:
- заполнение товаров по отбору (СКД);
- загрузку из внешнего файла (CSV/TSV/TXT) по штрихкоду;
- заполнение старой/расчётной/новой цены;
- расчёт по формуле;
- запись документа «Установка цен номенклатуры».

Можно скопировать содержимое файла и вставить в модуль формы обработки.

## Почему кнопка не появилась автоматически

`full_price_processing_module.bsl` — это код модуля формы, он **не создаёт элементы интерфейса сам по себе**.

Чтобы увидеть кнопку в обработке, в форме нужно вручную добавить команду и кнопку, привязав обработчик:

- команда `ЗаполнитьИзФайла` -> процедура `ЗаполнитьИзФайла(Команда)`;
- (или) команда `ЗаполнитьИзВнешнегоФайла` -> процедура `ЗаполнитьИзВнешнегоФайла(Команда)`.

После этого кнопка появится визуально и будет открывать выбор файла для загрузки.
