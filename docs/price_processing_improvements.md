# Доработка внешней обработки изменения цен

Ниже — готовый каркас доработки для вашей обработки: 

- в табличной части хранятся **старая цена**, **цена для расчёта** и **новая цена**;
- появляется заполнение не только по отбору, но и **из внешнего файла/буфера**;
- поддерживается массовая вставка **ШК + новая цена** (и опционально номенклатура/характеристика);
- после загрузки данные переносятся в основную ТЧ `Товары` по аналогии со штатной логикой 1С.

## 1) Что добавить в форму

1. Кнопку `ЗаполнитьИзФайла`.
2. Команду `ПеренестиИзБуфера` (если хотите сценарий «скопировал из Excel → вставил»).
3. Служебную ТЧ/таблицу формы `ИмпортТовары` с колонками:
   - `Штрихкод` (Строка)
   - `Номенклатура` (СправочникСсылка.Номенклатура)
   - `Характеристика` (СправочникСсылка.ХарактеристикиНоменклатуры)
   - `ЦенаНовая` (Число 15.2)
   - `ЦенаСтарая` (Число 15.2)
   - `ЦенаДляРасчета` (Число 15.2)
   - `Комментарий` (Строка)

> В основной ТЧ `Товары` должны быть колонки `Цена`, `ЦенаДляРасчета`, `ЦенаНовая`.

## 2) Серверная функция поиска по ШК

```bsl
&НаСервере
Функция НайтиНоменклатуруПоШК(Штрихкод) Экспорт
	Запрос = Новый Запрос;
	Запрос.Текст =
	"ВЫБРАТЬ ПЕРВЫЕ 1
	|	ШтрихкодыНоменклатуры.Номенклатура КАК Номенклатура,
	|	ШтрихкодыНоменклатуры.Характеристика КАК Характеристика
	|ИЗ
	|	РегистрСведений.ШтрихкодыНоменклатуры КАК ШтрихкодыНоменклатуры
	|ГДЕ
	|	ШтрихкодыНоменклатуры.Штрихкод = &Штрихкод";

	Запрос.УстановитьПараметр("Штрихкод", СокрЛП(Штрихкод));
	Выборка = Запрос.Выполнить().Выбрать();

	Если Выборка.Следующий() Тогда
		Возврат Новый Структура("Номенклатура,Характеристика", Выборка.Номенклатура, Выборка.Характеристика);
	КонецЕсли;

	Возврат Неопределено;
КонецФункции
```

## 3) Получение цен по номенклатуре/характеристике

```bsl
&НаСервере
Функция ПолучитьЦеныПоПозиции(Номенклатура, Характеристика) Экспорт
	ЦенаСтарая = 0;
	ЦенаДляРасчета = 0;

	Если ЗначениеЗаполнено(Объект.ЦенаДляИзменения) Тогда
		ЦенаСтарая = ПолучитьЦенуПоВидуЦены(Номенклатура, Характеристика, Объект.ЦенаДляИзменения);
	КонецЕсли;

	Если ЗначениеЗаполнено(Объект.ЦенаДляРасчета) Тогда
		ЦенаДляРасчета = ПолучитьЦенуПоВидуЦены(Номенклатура, Характеристика, Объект.ЦенаДляРасчета);
	КонецЕсли;

	Возврат Новый Структура("ЦенаСтарая,ЦенаДляРасчета", ЦенаСтарая, ЦенаДляРасчета);
КонецФункции

&НаСервере
Функция ПолучитьЦенуПоВидуЦены(Номенклатура, Характеристика, ВидЦены) Экспорт
	Запрос = Новый Запрос;
	Запрос.Текст =
	"ВЫБРАТЬ ПЕРВЫЕ 1
	|	ЦеныНоменклатурыСрезПоследних.Цена КАК Цена
	|ИЗ
	|	РегистрСведений.ЦеныНоменклатуры.СрезПоследних(&Дата, ) КАК ЦеныНоменклатурыСрезПоследних
	|ГДЕ
	|	ЦеныНоменклатурыСрезПоследних.Номенклатура = &Номенклатура
	|	И ЦеныНоменклатурыСрезПоследних.Характеристика = &Характеристика
	|	И ЦеныНоменклатурыСрезПоследних.ВидЦены = &ВидЦены";

	Запрос.УстановитьПараметр("Дата", Объект.Дата);
	Запрос.УстановитьПараметр("Номенклатура", Номенклатура);
	Запрос.УстановитьПараметр("Характеристика", Характеристика);
	Запрос.УстановитьПараметр("ВидЦены", ВидЦены);

	Выборка = Запрос.Выполнить().Выбрать();
	Если Выборка.Следующий() Тогда
		Возврат Выборка.Цена;
	КонецЕсли;

	Возврат 0;
КонецФункции
```

## 4) Загрузка табличных данных из Excel/файла

Ниже — универсальный серверный метод, который принимает `ТаблицаЗначений` (например, из формы предварительного заполнения, буфера обмена, либо из прочитанного табличного документа).

```bsl
&НаСервере
Процедура ЗагрузитьИмпортВТоварыНаСервере(ТаблицаИмпорта) Экспорт
	Если ТаблицаИмпорта = Неопределено Тогда
		Возврат;
	КонецЕсли;

	Для Каждого СтрокаИмпорта Из ТаблицаИмпорта Цикл
		Номенклатура = Неопределено;
		Характеристика = Неопределено;
		НоваяЦена = 0;

		Если СтрокаИмпорта.Свойство("ЦенаНовая") Тогда
			НоваяЦена = ?(ЗначениеЗаполнено(СтрокаИмпорта.ЦенаНовая), СтрокаИмпорта.ЦенаНовая, 0);
		КонецЕсли;

		Если СтрокаИмпорта.Свойство("Штрихкод") И ЗначениеЗаполнено(СтрокаИмпорта.Штрихкод) Тогда
			ДанныеШК = НайтиНоменклатуруПоШК(СтрокаИмпорта.Штрихкод);
			Если ДанныеШК <> Неопределено Тогда
				Номенклатура = ДанныеШК.Номенклатура;
				Характеристика = ДанныеШК.Характеристика;
			КонецЕсли;
		КонецЕсли;

		Если Номенклатура = Неопределено И СтрокаИмпорта.Свойство("Номенклатура") Тогда
			Номенклатура = СтрокаИмпорта.Номенклатура;
			Если СтрокаИмпорта.Свойство("Характеристика") Тогда
				Характеристика = СтрокаИмпорта.Характеристика;
			КонецЕсли;
		КонецЕсли;

		Если Не ЗначениеЗаполнено(Номенклатура) Тогда
			Продолжить;
		КонецЕсли;

		СтрокаТовары = Объект.Товары.Добавить();
		СтрокаТовары.Номенклатура = Номенклатура;
		СтрокаТовары.Характеристика = Характеристика;
		СтрокаТовары.Менять = Истина;

		Цены = ПолучитьЦеныПоПозиции(Номенклатура, Характеристика);
		СтрокаТовары.Цена = Цены.ЦенаСтарая;
		СтрокаТовары.ЦенаДляРасчета = Цены.ЦенаДляРасчета;
		СтрокаТовары.ЦенаНовая = НоваяЦена;

		Если СтрокаИмпорта.Свойство("Штрихкод") Тогда
			СтрокаТовары.ШК = СтрокаИмпорта.Штрихкод;
		КонецЕсли;
	КонецЦикла;
КонецПроцедуры
```

## 5) Клиентская кнопка «Заполнить из внешнего файла»

Вариант «как в заказе поставщику»: открыть форму-помощник, куда пользователь вставляет столбцы из Excel (ШК, цена), затем нажимает «Перенести».

```bsl
&НаКлиенте
Процедура ЗаполнитьИзФайла(Команда)
	// Открывайте свою форму загрузки (например, ОбщаяФорма.ЗагрузкаТоваровИзТабДок)
	ПараметрыФормы = Новый Структура;
	ПараметрыФормы.Вставить("Владелец", ЭтаФорма);
	
	ФормаЗагрузки = ПолучитьФорму("ОбщаяФорма.ЗагрузкаТоваровИзФайла", ПараметрыФормы, ЭтаФорма);
	ФормаЗагрузки.Открыть();
КонецПроцедуры
```

На форме загрузки:

- поле табличного документа;
- кнопка «Вставить из буфера»;
- сопоставление колонок (минимум: `Штрихкод`, `ЦенаНовая`);
- кнопка «Перенести», которая вызывает:

```bsl
Владелец.ЗагрузитьИмпортВТоварыНаСервере(ТаблицаИмпорта);
```

## 6) Важные проверки

Перед записью документа установки цен добавьте проверки:

1. `ЦенаНовая > 0`.
2. Номенклатура заполнена.
3. Если строка загружена по ШК, но номенклатура не найдена — писать в `Комментарий` в таблице импорта.
4. Исключать дубли по `(Номенклатура, Характеристика)` через `Свернуть` перед переносом.

## 7) Минимальная интеграция в вашу текущую логику

- Ваш метод `ЗаполнитьНаСервере()` оставить для сценария «по отбору».
- Новый метод `ЗагрузитьИмпортВТоварыНаСервере()` использовать для сценария «из Excel/файла/буфера».
- Далее общий этап `РассчитатьНаСервере()` и `ЗаписатьЦеныНаСервере()` остаётся прежним.
